# 前言

欢迎阅读 **《量化实战手册：从因子挖掘到 AI 交易》**。

这本书旨在构建一个从经典理论到现代工程实践，再到前沿 AI 技术的完整量化知识体系。无论你是刚入门的宽客，还是寻求技术突破的资深从业者，这里都有你需要的弹药。

## 本书结构

*   **第一部分：量化之道** - 回溯量化金融的历史，理解多空对冲的底层逻辑。
*   **第二部分：因子工厂** - 深入工程实践，掌握从数据清洗到回测归因的全套流水线。
*   **第三部分：因子军火库** - 系统梳理量价、基本面与另类因子。
*   **第四部分：前沿战法** - 探索 Transformer、Mamba 与 GNN 在量化中的应用。
