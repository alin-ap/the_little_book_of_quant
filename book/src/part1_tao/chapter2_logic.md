# 第 2 章：多空对冲逻辑

## Fama-French 因子计算方法

这些因子不是直接从股票软件里看的一个指标，而是通过**“构建多空对冲组合”**算出来的。

逻辑是：**买入表现强的一组，卖空表现弱的一组，两者收益的差值，就是这个因子的收益。**

为了计算准确，通常采用 **2x3 分组法**（把所有股票切两刀）：

### 1. 切蛋糕 (分组)
1.  **按市值 (Size) 切一刀：** 把全市场股票按市值排序，中间切开。
    * **S (Small):** 小市值组
    * **B (Big):** 大市值组
2.  **按账面市值比 (B/M) 切两刀：** 把股票按“账面市值比”（衡量价值）分成三份。
    * **H (High):** 价值股（便宜）
    * **M (Medium):** 中间
    * **L (Low):** 成长股（贵）

这样交叉组合，我们就得到了 2x3 = **6 个投资组合**（比如“小盘价值股”、“大盘成长股”等）。

### 2. 算因子 (做减法)

* **1. 市场因子 (Mkt-RF):**
    * `全市场回报率` - `无风险利率(如国债收益率)`
* **2. 规模因子 (SMB - Small Minus Big):**
    * 逻辑：做多小盘股，做空大盘股。
    * *计算：(3个小盘组合的平均收益) - (3个大盘组合的平均收益)*
    * $$SMB = \frac{1}{3}(Small Value + Small Neutral + Small Growth) - \frac{1}{3}(Big Value + Big Neutral + Big Growth)$$
* **3. 价值因子 (HML - High Minus Low):**
    * 逻辑：做多高账面市值比（价值股），做空低账面市值比（成长股）。
    * *计算：(2个价值组的平均收益) - (2个成长组的平均收益)*
    * $$HML = \frac{1}{2}(Small Value + Big Value) - \frac{1}{2}(Small Growth + Big Growth)$$
