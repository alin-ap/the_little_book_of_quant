## 🌟 核心理念：属于所有人的量化书

这不是一本由单一作者完成的书，而是一个**开源共建的量化知识库**。

量化交易是一个庞大且快速迭代的领域，没有谁能独自掌握所有真理。从经典的因子挖掘到最前沿的 Transformer/Mamba 模型，我们需要集结众人的智慧。

**这本书的作者不是我，而是屏幕前的每一个你。**

无论你是刚入门的学生，还是资深的从业者，只要你对量化有独到的见解、发现了一个拼写错误，或者想分享一段优雅的代码，都欢迎你提交 PR。让我们一起构建这本属于社区的实战手册。

## 🤝 如何参与 (Contributing)

我们非常欢迎并感谢任何形式的贡献！每一行提交的代码或文字，都是在为量化社区添砖加瓦。

- **纠错**：发现错别字、逻辑漏洞或公式错误？直接提 PR 修复。
- **补充**：觉得某个知识点讲得不够透？欢迎补充内容、图表或代码案例。
- **分享**：有新的 SOTA 模型复现、独特的因子思路或实战心得？请新开章节分享给大家。

### 参与步骤

1. **Fork** 本仓库到你的 Github 账号。
2. **Clone** 到本地：
   ```bash
   git clone https://github.com/your-username/the_little_book_of_quant.git
   ```
3. **修改** 内容（Markdown 格式）。
4. **提交** Pull Request (PR)。我们会尽快 Review 并合并。

## 🛠️ 本地预览 (Build Locally)

如果你想在本地查看修改效果，可以使用 [mdBook](https://rust-lang.github.io/mdBook/) 运行。

```bash
# 1. 安装 mdBook (MacOS)
brew install mdbook
# Windows/Linux 用户请参考 mdBook 官方文档

# 2. 启动本地服务
mdbook serve --open
```

运行后，浏览器将自动打开 `http://localhost:3000`。
